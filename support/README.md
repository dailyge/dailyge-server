## ⛺️ Support 모듈

Support 모듈. 로깅, 모니터링 등 API 서버를 지원하는 역할을 담당합니다.

<br/><br/><br/>

Storage 모듈은 다른 모듈에 의존하지 않습니다.

| API | Scheduler | Storage | Support |
|:---:|:---------:|:-------:|:-------:|
|  X  |     X     |    X    |    -    |

&nbsp;&nbsp; - API: API 모듈 <br/>
